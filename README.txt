!NOTICE!

Audio/midi device list start from number 0.

e.g. Audio device list
device 0 Apple Inc.: Built-in Microphone
device 1 Apple Inc.: Built-in Output
device 2 ma++ ingalls for Cycling '74: Soundflower (2ch)
device 3 ma++ ingalls for Cycling '74: Soundflower (64ch)
device 4 RME-Audio: Fireface UCX (23507154)

e.g. Midi Device list
0: Fireface UCX (23507154) Port 1
1: Fireface UCX (23507154) Port 2


if you want to use Built-in Output for your audio output
press 4

if you want to use Fireface UCX (23507154) Port 2 for your Midi port
press Shift + 1
